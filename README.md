# AutoTorch 🕯️

**AutoTorch** is a powerful Minecraft Bedrock Edition add-on designed to automate the lighting process in your world. Say goodbye to manually placing torches in every dark corner—this add-on does it for you intelligently and efficiently.

## ✨ Key Features

* **Smart Placement:** Automatically detects low light levels and places torches on walls or the floor.
* **Highly Customizable:** Adjust settings such as placement priority (wall vs. floor) and darkness threshold directly in-game.
* **Intuitive GUI:** Manage your settings through a clean interface accessible by sneaking while holding a torch.
* **Multi-language Support:** Built-in localization support (RU, EN, UK, ZH, JA) with aesthetic flag icons.

## 🎮 How to Use

1.  **Open Settings:** Hold a torch in your main hand and **sneak**.
2.  **Chat Commands(They may not work):**
    * `/torch menu` — Open the graphical settings menu.
    * `/torch on` — Enable the AutoTorch module.
    * `/torch off` — Disable the AutoTorch module.


## 🛠 Technical Details

Built on the Minecraft Script API.
* **Game Version:** 26.20+
* **Dependencies:** `@minecraft/server`, `@minecraft/server-ui`
